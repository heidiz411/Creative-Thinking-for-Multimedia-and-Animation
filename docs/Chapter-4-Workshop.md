### **2. ขั้นตอนการปฏิบัติงาน (Step-by-Step Instructions)**

#### **ขั้นตอนที่ 1: ตั้งต้นด้วยมนุษย์ — Character Core (25 นาที)**

เลือกตัวละคร 1 ตัวจากกิ่ง **Main Characters** ใน Mind Map ของคุณ แล้วกรอกตารางด้านล่าง **ห้ามเปิด AI ในขั้นตอนนี้** จากนั้นวาด thumbnail เร็ว 3 ช่อง โดยบังคับให้แต่ละช่องใช้รูปทรงเด่นต่างกัน (วงกลม = เป็นมิตร / สี่เหลี่ยม = มั่นคง / สามเหลี่ยม = อันตราย-ว่องไว) แล้ววงกลมเลือกแบบที่ตรงกับนิสัยตัวละครที่สุด

| หัวข้อ | คำตอบของนักศึกษา |
|---|---|
| ชื่อตัวละคร (จาก Mind Map) | |
| บทบาทในเรื่อง / Archetype (ฮีโร่, ผู้ช่วย, ตัวป่วน ฯลฯ) | |
| เป้าหมายของตัวละคร (โยงกับกิ่ง Key Conflicts) | |
| นิสัยขัดแย้งในตัวเอง 1 อย่าง (เช่น กล้าหาญแต่ขี้ลืม) | |
| รูปทรงเด่นที่เลือก + เหตุผล | |
| สี 3 สี (หลัก/รอง/เน้น) + สื่ออารมณ์อะไร | |

#### **ขั้นตอนที่ 2: Silhouette Test (15 นาที)**

ระบายแบบที่เลือกเป็น **เงาดำล้วน 2 ท่าทาง** (ท่ายืน + ท่าประจำตัว) ลงกระดาษหรือแท็บเล็ต จากนั้นจับคู่กับเพื่อน สลับกันทายว่า "ตัวละครนี้นิสัยยังไง" โดยห้ามอธิบายก่อน

> _เพื่อนทายว่า:_ ................................................... _ตรงกับที่ตั้งใจหรือไม่:_ [ ] ตรง [ ] ไม่ตรง — _ถ้าไม่ตรง จะแก้ตรงไหน:_ ...................................................

#### **ขั้นตอนที่ 3: World Snapshot (15 นาที)**

สรุป "โลก" ของคุณจากกิ่ง **World & Environment** และ **Key Conflicts** ให้เหลือครึ่งหน้า เพื่อใช้เป็นข้อมูลป้อน AI ในขั้นตอนถัดไป

| หัวข้อ | คำตอบ (1–2 บรรทัด) |
|---|---|
| สถานที่เด่น 1 แห่งที่ตัวละครใช้ชีวิตอยู่ | |
| กฎของโลก 1 ข้อ (สิ่งที่ทุกคนในโลกนี้ต้องทำ/ห้ามทำ) | |
| ตัวละครของฉันมีบทบาท/อาชีพอะไรในโลกนี้ | |
| สไตล์ภาพจากกิ่ง Unique Visual Style (เช่น สีนีออน, ลายเส้นการ์ตูน) | |

#### **ขั้นตอนที่ 4: AI Prompt Lab — สร้างภาพอ้างอิง 3 รอบ (40 นาที)**

เปิดเครื่องมือสร้างภาพ AI (Adobe Firefly, Midjourney หรือเครื่องมือที่อาจารย์กำหนด) แล้วเขียน Prompt ตามสูตร 6 ส่วน:

> **Subject (ตัวละคร+ท่าทาง) + Style (จากกิ่ง Visual Style) + Lighting/Mood (จากมู้ดแอนด์โทนใบงานที่ 2) + มุมกล้อง + สี 3 สีจากขั้นตอนที่ 1 + คำขยายคุณภาพ**

**ตัวอย่าง (ตัวเลือก A):** _"a young fish-noodle vendor character, floating market boat, cyberpunk style, neon pink and teal lighting, night scene, full body, concept art"_

| รอบที่ | Prompt เต็มที่ใช้ | ผลที่ได้ / ปัญหา | สิ่งที่จะแก้รอบถัดไป |
|---|---|---|---|
| 1 | | | |
| 2 | | | |
| 3 | | | |

**กติกาจริยธรรม (เหมือนใบงานที่ 2):** ห้ามใส่ชื่อศิลปิน บุคคลจริง หรือตัวละครมีลิขสิทธิ์ใน prompt / ระบุชื่อเครื่องมือและเวอร์ชันที่ใช้: ................................. / ภาพ AI ใช้เป็น reference เท่านั้น ไม่ใช่ชิ้นงานสำเร็จ

#### **ขั้นตอนที่ 5: Mini Gallery และส่งงาน (15 นาที)**

นำภาพรอบที่ดีที่สุดขึ้นจอ/ติดผนังพร้อม silhouette เพื่อนโหวตสติกเกอร์ 1 ดวงให้ผลงานที่ "ภาพกับนิสัยตัวละครตรงกันที่สุด" (ห้ามโหวตตัวเอง) จากนั้นส่งใบงาน + ไฟล์ภาพเข้าระบบก่อนออกจากห้อง
